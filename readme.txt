This is a plugin that helps your Nagios client talk to Wordpress to find out if you have any available updates.

== Description ==

This is a plugin that contains some scripts that Nagios can poll to see if you have an Wordpress Updates available. 

The benefit being that if you monitor your systems using Nagios or a Nagdash dashboard you will see straight away when there are updates available rather than waiting until you log in. 

More Info: https://github.com/fredbradley/nagios-wordpress-updates-checker/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/nagios-wordpress-updates-checker` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress
